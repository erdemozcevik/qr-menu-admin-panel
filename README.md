# QR Menu Admin Panel


An admin-based QR menu management panel built with React and Tailwind CSS.

## Features
- Add new categories
- Add products under categories
- Delete products
- Update product prices
- Dynamic menu preview
- Responsive admin panel layout

## Technologies
- React
- Vite
- Tailwind CSS
- JavaScript

## Installation

```bash
npm install
npm run dev
```
